## Topics Covered

* Neural coding and information representation
* Relationship between models and data
* Perceptual illusions as computational problems
* Linking brain activity to behavioral outputs

---

## Learning Objectives

By the end of this session, participants should be able to:

* Explain how information is encoded in neural systems
* Distinguish between model-based and data-driven approaches
* Understand how computational models explain perception
* Apply conceptual frameworks to real neuroscience examples

##  Session Flow (What to Expect)

* Guided understanding of model vs data perspectives
